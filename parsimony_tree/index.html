Go Parsimony Tree is an app that takes a character matrix as input and uses it to create a maximum parsimony tree using the phangorn package in R (Schliep 2011). It returns a VEGA visualization of the maximum parsimony tree and an image of the most parsimonious reconstruction for a character of interest.

1. Upload your character matrix (.csv). This application assumes that the first column contains the tip labels, and the first row is the outgroup.

2. Choose a character (column of the matrix) to use for the most parsimonious reconstruction output.

3. Click GO.



Source cited: Schliep K.P. 2011. phangorn: phylogenetic analysis in R. Bioinformatics, 27(4) 592-593
